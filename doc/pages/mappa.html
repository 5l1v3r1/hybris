Un array associativo, ovvero una collezione nella quale ogni valore è associato ad una chiave. Data questa associazione “chiave → valore”, per accedere agli elementi della collezione non si userà un indice numerico posizionale come nel caso degli array, bensì verrà usata la chiave stessa.

Può essere inizializzata con dei valori tramite operatore [ ] : 

 m = [ "zero" : 0, "uno" : 1, "due" : 2 ];

Oppure, qual'ora fosse necessario crearla inizialmente vuota : 

 m = [:];

Metodi

Questa tipologia ha dei metodi intrinsechi che permettono varie operazioni su di essa.

size()

Restituisce il numero di elementi che contiene la mappa. 

println( [ "zero" : 0, "uno" : 1, "due" : 2 ].size() );
3

pop()

Rimuove l'ultimo elemento dalla mappa e ne restituisce un istanza. 

println( [ "zero" : 0, "uno" : 1, "due" : 2 ].pop() );
2

unmap( key )

Rimuove l'elemento mappato con la chiave 'key' e ne restituisce un istanza. 

println( [ "zero" : 0, "uno" : 1, "due" : 2 ].unmap("uno") );
1

has( key )

Restituisce true se la mappa contiene un elemenento mappato con la chiave 'key', altrimenti restituisce false . 

println( [ "zero" : 0, "uno" : 1, "due" : 2 ].has( "uno" ) );
 
println( [ "zero" : 0, "uno" : 1, "due" : 2 ].has( "tre" ) );
true
false

keys()

Restituisce un array contenente le chiavi della mappa. 

println( [ "zero" : 0, "uno" : 1, "due" : 2 ].keys() );
array {
      zero
      uno
      due
}

values()

Restituisce un array contenente i valori della mappa. 

println( [ "zero" : 0, "uno" : 1, "due" : 2 ].values() );
array {
      0
      1
      2
}

Esempio

import std.*;
 
// creo una mappa con elementi e chiavi di tipologia mista e la stampo
m = [ "zero" : 0, 
      "uno" : 2 - 1, 
      "due" : 2.0, 
      "tre" : [ 0, 1, 2, 3 ] ];
print(m);
 
// stampo l'elemento con chiave 'due'
println( m["due"] );
 
// in questo contesto, 144 non è considerato un indice come nel caso degli array, ma una chiave a sua volta
// di conseguenza in questo modo sto mappando la stringa "lulz" sulla chiave 144 
m[144] = "lulz";
print(m);
 
// rimuovo l'elemento mappato con la chiave "uno"
m.unmap("uno");
print(m);
 
// anche un decimale può essere usato come chiave
m[12.4] = "dodici virgola quattro";
 
// così come un elemento della mappa può essere una mappa a sua volta
m["lol"] = [ "Ciao" : "mao", "Bau"  : [ m ] ];
print(m);
 
/* nel caso delle mappe, il costrutto foreach ha la seguende sintassi :
		foreach( <chiave> : <valore> of <mappa> )	
*/					
foreach( key -> value of m ){
	// se il valore è una mappa o un array, stampo la sua struttura
	if( isarray(value) || ismap(value) ){
		print( key + " : " );
		print(value);
	}
	// se altrimenti è un tipo semplice, stampo direttamente il suo valore
	else{
		print( key + " : " + value + "\n" );	
	}
}
