Una struttura è un tipo di dato composto da vari attributi che la rappresentano, dichiarato tramite la keywork struct .

Per dichiarare una struttura, la sintassi è : 

struct /* nome della struttura */ {
// attributi
}

Ad esempio : 

struct Persona {
    nome;
    cognome;
    età;
}

Che può essere dichiarata analogamente, con gli attributi su una sola riga : 

struct Persona {
    nome, cognome, età;
}

Per creare una variabile ed impostarla con una struttura da noi definita, si usa l'operatore new seguito dal nome della struttura : 

p = new Persona();

Possiamo inoltre passare dei valori a questo costrutto, valori che saranno utilizzati per inizializzare gli attributi della struttura stessa : 

p = new Persona( "Mario", "Rossi", 41 );

Per accedere agli attributi della struttura, si usa l'operatore . : 

println( "Il Sig. " + p.nome + " " + p.cognome + " ha " + p.età + " anni." );
