Il costrutto do/while, analogamente al costrutto while, esegue un insieme di operazioni racchiuse tra parentesi graffe, fino a quando la condizione booleana indicata è vera, con la differenza che la condizione viene controllata ad ogni loop, dopo che il blocco di codice è stato eseguito .

La sintassi è : 

do{
    // azioni
}
while( /* condizione */ );

Ad esempio : 

n = 10;
do{
    println(n);
    n--;
}
while( n >= 0 );

Eseguirà la funzione println fino a quando la variabile n, che viene ad ogni ciclo decrementata, è maggiore o uguale a zero.
