Una lista ordinata di valori di tipologia non necessariamente omogenea, caratterizzata da una grandezza rappresentante il numero dei suoi elementi ai quali si può accedere tramite l'operatore subscript ed un indice numerico che ne rappresenti la posizione all'interno della collezione stessa.

Può essere inizializzato con dei valori tramite operatore [ ] : 

 a = [ 1, 2, "foo", "bar" ];

Metodi

Questa tipologia ha dei metodi intrinsechi che permettono varie operazioni su di essa.

size()

Restituisce il numero di elementi che contiene l'array. 

println( [ "foo", "bar", "meh" ].size() );
3

pop()

Rimuove l'ultimo elemento dell'array e ne restituisce un istanza . 

println( [ "foo", "bar", "meh" ].pop() );
meh

remove( index )

Rimuove un elemento dall'array ad un indice specifico e ne restituisce un istanza. 

println( [ "foo", "bar", "meh" ].remove(1) );
bar

contains( object )

Restituisce true se l'array contiene l'elemento specificato, altrimenti false . 

println( [ "foo", "bar", "meh" ].contains( "foo" ) );
 
println( [ "foo", "bar", "meh" ].contains( "moo" ) );
true
false

join( glue )

Restituisce una stringa con tutti gli elementi dell'array concatenati tramite il carattere o la stringa 'glue'. 

println( [ "foo", "bar", "meh" ].join( ", " ) );
foo, bar, meh

Esempio

import std.*;
 
// inizializzo un array con valori di tipi misti, uno dei quali è un array a sua volta
a     = [ 0, 
	  1 + 0 * 1, 
	  [ 2, "due", 2 + 0 ], 
	  3 ];
// la variabile intera 'elems' ora contiene il numero di elementi dell'array 'a'
elems = a.size();
 
println( "L'array 'a' ha " + elems + " elementi :\n" );
// stampo ogni elemento di 'a', nel caso dell'array all'indice 2 verrà stampata la sua struttura 
for( i = 0; i < elems; i++ ){
	print( "elemento " + i + " : " );
	println( a[i] );	
}
 
// rimuovo il secondo elemento da 'a' e stampo nuovamente il contenuto
println( "\na - 2° elemento : \n" );
a.remove( 1 );
elems = a.size();
for( i = 0; i < elems; i++ ){
	print( "elemento " + i + " : " );
	println( a[i] );	
}
 
/* eseguo un loop con il costrutto 'foreach' dove la sintassi è :
		foreach( <elemento> of <array> )
*/
println( "\nTest foreach : \n" );
foreach( item of a ){
	print("item = ");
	print(item);	
	// stampo il tipo di ogni elemento
	println( "\t" + typeof(item) );
}
 
// rimuovo gli elementi residui di a
println( "\nSvuoto 'a' :\n" );
elems = a.size();
for( i = 0; i < elems; i++ ){
	a.pop();
	println(a);	
}
