Il costrutto switch serve per confrontare una variabile con una serie di valori ed eseguire solo il blocco di codice associato al valore al quale effettivamente corrisponde o, nel caso viene specificato un blocco default, eseguire quello nel caso non ci siano corrispondenze tra i valori elencati.

La sintassi Ã¨ : 

switch( variabile da confrontare ){
   case valore :
       // operazioni
   break;
 
   case altro valore :
       // operazioni
   break;
 
   case altro valore ancora :
       // operazioni
   break;
 
   default :
      // operazioni se non sono state trovate corrispondenze
}

Ad esempio 

word = "ciao";
 
switch(word){
	case 1 : 
		word .= 1;
	break;
 
	case "lol" :
		println( "hai detto lol" );
	break;
 
	case "ciao" :
		println( "hai detto ciao" );
	break;
 
	default: 
		println( "non so cosa hai detto" );
}
