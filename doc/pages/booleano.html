Quella booleana, è una tipologia che può assumere i valori “true” o “false”.

Esempio : 

function disparo( n ){
    if( (n % 2) != 0 ){
        return true;
    }
    else{
        return false;
    }
}
 
foreach( n of 1..100 ){
    if( disparo(n) == true ){
        println( n + " è disparo." );
    }
}
