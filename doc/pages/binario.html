Il tipo binary serve a gestire e rappresentare dati di tipo binario in una collezione. Analogamente all'array si può accedere agli elementi tramite l'operatore subscript e ad un indice che ne rappresenti la posizione.

Esempio : 

b = binary( 10, 12, 0xFA, 123, 'c' );
println("b è grande ".sizeof(b)." bytes." );
foreach( c of b ){
    println(c);
}
for( i = 0; i < sizeof(b); i++ ){
    println( "b[".i."] = ".toint(b[i])." (".b[i].")");
}

Tramite la funzione pack è possibile trasformare una variabile di qualsiasi tipologia in un binary, specificando quanti byte utilizzare nella variabile originale per effettuare la conversione. 

import std.*;
 
stringa = "Ciao Mondo!";
binario = pack( stringa, 11 ); 
 
println( binario );

Questo esempio stamperà l'output 

binary {
  4369616F204D6F6E646F21
}
