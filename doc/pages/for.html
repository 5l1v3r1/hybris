Il ciclo for serve ad eseguire un blocco di istruzioni un determinato numero di volte o finchè la condizione specificata è vera.

La sintassi è : 

for( /* inizializzazione */; /* condizione */; /* variazione */ ){
    // operazioni
}

Ad esempio, per eseguire una stampa 10 volte, si può scrivere : 

for( i = 0; i < 10; i++ ){
    println(i);
}
