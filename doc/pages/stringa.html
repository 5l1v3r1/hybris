Una stringa è una collezione di caratteri che può contenere un numero teoricamente infinito di elementi, elementi che occupano 1 byte ciascuno, nei limiti della memoria del computer. E' caratterizzata, come ogni collezione, da una grandezza rappresentante il numero di elementi 1) e dall'operatore subscript per l'accesso tramite indice agli elementi che la compongono 2).

Metodi

Questa tipologia ha dei metodi intrinsechi che permettono varie operazioni su di essa.

length()

Restituisce il numero di caratteri che contiene la stringa . 

println( "hello world".length() );
11

find( str )

Cerca una stringa o un carattere all'interno della stringa principale, restituisce l'indice dell'elemento trovato o false in caso non ci sia alcun match. 

s = "hello world";
 
println( s.find("wor") );
 
println( s.find("foo") );
6
false

substr( index, length (opzionale) )

Restituisce una sottostringa lunga 'length' caratteri, partendo dall'indice 'index' . Se il parametro length non è specificato, restituisce una sottostringa dall'indice fino alla fine della stringa principale . 

s = "hello world";
 
println( s.substr(6) );
 
println( s.substr(6,4) );
world
worl

substr( find, replace )

Restituisce una stringa tutte le occorrenze di 'find' nella quale sono state sostituite con 'find'. 

s = "hello world";
 
println( s.replace( "world", "hybris" ) );
hello hybris

split( separator )

Divide la stringa in un array di stringhe, utilizzando il separatore specificato. 

s = "hello world from hybris";
 
println( s.split( " " ) );
array {
      hello
      world
      from
      hybris
}

trim()

Elimina ogni carattere “white space” (tab, new line, line feed, ecc) dall'inizio e dalla fine della stringa, restituendo una nuova stringa . 

s = "hello world from hybris";
 
println( "\t\n\r hello world\n".trim() );
hello world

Esempio

 // dichiaro una stringa
 stringa = "Ciao mondo !";
 // ne calcolo la lunghezza inserendola nell intero
 lunghezza = stringa.length();
 // stampo la stringa
 println( "Stringa originale : ’" + stringa + "‘" );
 // nel ciclo, la variabile ’i' viene contemporaneamente dichiarata ed inizializzata
 for( i = 0; i < lunghezza; i++ ){
     // imposto ogni carattere con il valore valore i + ’a’
     stringa[i] = i + 'a';
 }
 // stampo la stringa nuovamente
 println( "Stringa modificata : ’" + stringa + "‘" );
Stringa originale : ’Ciao mondo !‘
Stringa modificata : ’abcdefghijkl‘

2) Un tentativo di accesso ad un indice al di fuori della sua grandezza genererà l'errore index out of bounds
