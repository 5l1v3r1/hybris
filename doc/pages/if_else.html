Il costrutto if serve ad eseguire una determinata porzione di codice, racchiusa tra parentesi graffe, se e solo se si verifica una data condizione booleana. Può essere (opzionalmente) accompagnato dal costrutto else il quale, nel caso in cui la prima condizione booleana fosse falsa, eseguirà invece il blocco sottostante.

Vediamone la sintassi : 

 if( condizione ){
   // azioni
 }
 else{
   // altre azioni
 }

Ad esempio : 

numero = 10;
if( numero >= 2 ){
   println( "Il numero è uguale o maggiore di due ." );
}
else{
   println( "Il numero è minore di due ." );
}

In questo caso è facile intuire che la sintassi è analoga a quella dei maggiori linguaggi di programmazione e così i vari operatori booleani di confronto ==, !=, >, >=, <, ecc.
