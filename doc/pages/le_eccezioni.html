Un eccezione è un evento generato da una determinata operazione, generamente associato ad un errore (ma non necessariamente), che può essere “lanciato” tramite la direttiva throw, ad esempio : 

foreach( i of 1..100 ){
	println( "Dopo " + i );
 
	if( i == 23 ){
	    throw "Siamo arrivati a 23!";
	}
}

Tuttavia, se il seguente codice venisse eseguito, genererebbe l'errore : 

ERROR : Unhandled exception : Siamo arrivati a 23 !

Questo perchè, come suggerisce l'errore stesso, l'eccezione non è stata gestita, ovvero non è stato implementato nello script un metodo per istruire l'engine su cosa fare quando viene generata l'eccezione. Per fare questa, si usano le direttive try, che deve racchiudere il pezzo di codice che potenzialmente può generare un eccezione, e catch che verrà eseguito se e quando un eccezione viene generata, ad esempio : 

try{
	foreach( i of 1..100 ){
		println( "Dopo " + i );
 
		if( i == 23 ){
			throw "Siamo arrivati a 23!";
		}
	}
}
catch( e ){
	println( e );
}

Genererà correttamente l'output : 

...
...
Siamo arrivati a 23!

E' inoltre possibile specificare un blocco di istruzioni da eseguire in ogni caso, che l'eccezione venga generata o meno, alla fine traimte la direttiva finally : 

try{
	foreach( i of 1..100 ){
		println( "Dopo " + i );
 
		if( i == 23 ){
			throw "Siamo arrivati a 23!";
		}
	}
}
catch( e ){
	println( e );
}
finally {
        println( "Questa operazione viene eseguita alla fine." );
}

Che genererà : 

...
...
Siamo arrivati a 23!
Questa operazione viene eseguita alla fine.

La direttiva throw è in grado di lanciare un eccezione di qualsiasi tipo, in questi esempi abbiamo visto come lanciare un eccezione di tipo stringa, tuttavia la tipologia è indifferente, sarà poi il blocco all'interno della direttiva catch eventualmente a gestire l'eccezione in base al tipo che ci si aspetta.

Per standardizzare le eccezioni, è possibile utilizzare la classe std.Exception, ad esempio : 

include std.Exception;
 
try{
	foreach( i of 1..100 ){
		println( "Dopo " + i );
 
		if( i == 23 ){
			throw new Exception("Siamo arrivati a 23!");
		}
	}
}
catch( e ){
	println( e );
}

O alternativamente, utilizzando il costruttore tramite il quale specificare sia il file che la riga nei quali è stata generata l'eccezione : 

include std.Exception;
 
try{
	foreach( i of 1..100 ){
		println( "Dopo " + i );
 
		if( i == 23 ){
			throw new Exception( __FILE__, __LINE__, "Siamo arrivati a 23!");
		}
	}
}
catch( e ){
	println( e );
}

Che stamperà : 

...
...
[exception.hy:6] Siamo arrivati a 23!
