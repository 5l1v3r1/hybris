Questo tipo di operatore restituisce un valore booleano che può essere “vero” o “falso” (o 1 e 0 che dir si voglia) e si utilizza per controllare determinate condizioni, ad esempio nei costrutti condizionali e nei costrutti iterativi di Hybris .

!a
Restituisce il valore negato di 'a'.
a == b
Controlla se 'a' è uguale a 'b'.
a != b
Controlla se 'a' è diverso a 'b'.
a < b
Controlla se 'a' è minore a 'b'.
a > b
Controlla se 'a' è maggiore a 'b'.
a ⇐ b
Controlla se 'a' è minore o uguale a 'b'.
a >= b
Controlla se 'a' è maggiore o uguale a 'b'.
a || b
Controlla se 'a' o 'b' hanno un valore booleano positivo.
a && b
Controlla se 'a' e 'b' hanno un valore booleano positivo.
