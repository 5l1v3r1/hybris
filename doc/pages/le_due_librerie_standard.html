Hybris comprende un set di funzioni integrate nella sua libreria standard, suddivise in moduli dinamici raggruppati in namespace in base al loro campo di utilizzo.

Per caricare uno di questi moduli e utilizzarne le funzionalità, è necessario utilizzare la direttiva import all'interno di uno script, ad esempio, uno script che utilizza la funzione println della libreria standard, riporterà nella prima riga : 

import std.io.console;

Alternativamente, se si volesse importare un intero namespace, è possibile utilizzare insieme alla direttiva import l'operatore '*', ad esempio : 

import std.io.*;

Importerà tutti i moduli contenuti nel namespace std.io.

La libreria standard è composta dai seguenti moduli :

  • std
    • math : Funzionalità matematiche
    • pcre : Supporto alle espressioni regolari
    • encoding : Funzionalità di encoding
  • io
    • console : I/O su console
    • file : I/O su file
    • xml : I/O su flussi XML
    • network
      • tcp : Primitive per i socket tcp/ip
      • http : Funzionalità HTTP e HTTPS
  • lang
    • string : Gestione stringhe
    • array : Gestione array
    • matrix : Gestione matrici
    • binary : Gestione flussi binari
    • map : Gestione mappe associative
    • type : Gestione dei tipi
    • reflection : Introspezione dei tipi
  • os
    • dll : Caricamento ed esecuzione di funzioni da dll di sistema
    • threads : Multithreading
    • time : Gestione del tempo e della data
    • process : Gestione multi processo
  • hashing
    • crc32 : Calcolo del checksum crc32
    • md5 : Calcolo dell'hash md5
    • sha1 : Calcolo dell'hash sha1
    • sha2 : Calcolo dell'hash sha2 (sia 256 che 224)
