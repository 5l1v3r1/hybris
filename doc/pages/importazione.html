Hybris comprende un set di funzioni integrate nella sua libreria standard, suddivise in moduli dinamici raggruppati in namespace in base al loro campo di utilizzo.

Per caricare uno di questi moduli e utilizzarne le funzionalità, è necessario utilizzare la direttiva import all'interno di uno script, ad esempio, uno script che utilizza la funzione println della libreria standard, riporterà nella prima riga : 

import std.io.console;

Alternativamente, se si volesse importare un intero namespace, è possibile utilizzare insieme alla direttiva import l'operatore '*', ad esempio : 

import std.io.*;

Importerà tutti i moduli contenuti nel namespace std.io.

Per maggiori informazioni sui moduli che compongono la libreria standard, fare riferimento all'apposita sezione.
