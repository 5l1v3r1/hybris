Il ciclo foreach funge come abbreviazione del ciclo for quando si deve operare velocemente su variabili di tipo array o mappa e accedere agli elementi della loro collezione senza dover specificare un indice 1).

La sintassi (nel caso di una variabile di tipo array) Ã© : 

foreach( /* elemento */ of /* array */ ){
    // operazioni
}

Ad esempio, per stampare ogni elemento di un array : 

foo = array( 1, 2, 3, 4, 5, 6 );
foreach( item of foo ){
    println(item);
}

Se invece volessimo stampare ogni chiave e relativo elemento di una mappa : 

foo = map( "nome"    : "Mario",
           "cognome" : "Rossi" );
 
foreach( chiave -> elemento of foo ){
     println( chiave + " -> " + elemento );
}
1) In questo ciclo, la variabile viene impostata per copia e non per riferimento, quindi cambiarla non significa cambiare anche l'elemento della collezione.
Se si desidera accedere in scrittura agli elementi della collezione si deve usare il ciclo for con un indice.
