Per run time loading (caricamento a runtime), si intende caricare uno script o una stringa contenente del codice Hybris durante l'esecuzione, tramite la funzione load (per gli script) e la funzione eval (per le stringhe).

Ad esempio : 

import std.*;
 
load( "uno_script" );

o : 

import std.*;
 
eval( "foo = 123;" );
 
println( foo );

La differenza sostanziale con le direttive di inclusione e di importazione, è che queste due funzioni non vengono eseguite indiscriminatamente durante la fase di parsing, bensì possono essere utilizzate come qualsiasi altra funzione del linguaggio e di conseguenza il loading o il parsing di codice esterno può essere effettuato selettivamente dall'utente.
