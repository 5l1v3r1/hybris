E' possibile includere script esterni tramite la direttiva include, ad esempio : 

include funzioni.hy;

oppure 

include funzioni;

Includerà lo script “funzioni.hy” .

I file vengono cercati prima nella cartella dello script principale, nel caso non venissero trovati viene considerata la cartella di default :

/usr/lib/hybris/include/

Nella quale risiete la libreria standard ad oggetti.

Nota : La direttiva include viene eseguita durante la fase di parsing dello script, di conseguenza non è possibile utilizzarla a runtime. Ad esempio, lo script : 

if( variabile == 1 ){
    include uno_script;
}
else{
    include altro_script;
}

Includerà entrambi gli script a prescindere dalle condizioni, se si vuole caricare a run time selettivamente un file, bisogna utilizzare la funzione load, per questo fare riferimento alla sezione Run Time Loading.
