Il costrutto ternario è un abbreviazione del costrutto if/else che si utilizza nel caso in cui bisogna impostare una variabile con un determinato valore in base ad una condizione booleana.

La sintassi è : 

( condizione ? /* valore nel caso true */ : /* valore nel caso false */ )

Ad esempio : 

numero   = 10;
risposta = ( numero >= 2 ? "si" : "no" );
 
println( "Il numero ".numero." è maggiore o uguale a 2 ? ".risposta );
